# MPD nowplaying
Just a very simple minimalistic mpd widget/player that shows album art.
### Controls:

**Q:** Quit

**P/Space/Left-click:** Toggle play/pause

## Preview
![preview of the the widget](screenshots/preview.png "preview")

## How to build

Dependencies:
 - mpd
 - libmpdclient
 - odin

```bash
$ odin build . -build-mode:exe
```
